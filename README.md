# Mactokio

Native macOS menu bar authenticator for one-time codes (TOTP/HOTP).

## Features

- Menu bar app with quick access to codes
- Support for TOTP and HOTP algorithms
- Import secrets from any provider (Google Authenticator, Microsoft Authenticator, etc.)
- Add accounts via secret key or QR code
- Countdown timer for code expiration
- Copy code to clipboard with one click

## Requirements

- macOS 13.0+

## Build

```bash
make build
```

## Run

```bash
make run
```

## Release

```bash
make release
```
